TripCo, Trip Calculator

Project

This project was made in my Software Engineering class at CSU. At the beginning of the class we formed teams of three to make a project with for the rest of the semester. Using Java and Git/Github we made a program that would take a CSV file with a list destinations in Colorado and create an XML itinerary and an SVG map showing a route to take, connecting each destination. As the project went on, it was adapted to take input from an SQL database containing thousands of locations across the world and output a KML file that is compatible with Google Earth. The user could choose what exact destinations they wanted using a GUI and even use optional constraints to find the shortest trip.
